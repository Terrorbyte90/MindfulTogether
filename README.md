# MindfulTogether 💜

En community-app för personer med psykisk ohälsa. Profiler, kunskap och stödcommunities.

## 📱 Funktioner

### Hem
- Välkomstskärm med snabbåtkomst till kris-hjälp
- Aktiva forum-trådar

### Community
- Forum för att dela och få stöd
- Kategorier: Ångest, Depression, Stress, Utbrändhet, Relationer, Självskada, Sorg, Tips & Råd
- Skapa egna inlägg
- Svara och hjälpa andra

### Kunskap
- Artiklar sammanfattade från forum
- Sök och filtrera efter kategori
- Markera som hjälpsam

### Mående
- Dagligt mood-check (emoji-scale)
- Anteckningar
- Historik

### Profil
- Anonymt som standard
- Statistik (inlägg, hjälpsam, ämnen)
- Inställningar

## 🚀 Kom igång

```bash
# Klona
git clone https://github.com/Terrorbyte90/MindfulTogether.git
cd MindfulTogether

# Installera xcodegen
brew install xcodegen

# Generera projekt
xcodegen generate

# Öppna i Xcode
open MindfulTogether.xcodeproj
```

## 🎨 Design

- Färger: Lavendel, sagegrönt, mjuka blå toner
- Tillgänglighet: Stor text, hög kontrast
- Integritet: Anonymt som standard
